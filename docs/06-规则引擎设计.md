## 6.1 规则类型体系

规则引擎是本应用的核心组件，负责执行确定性校验规则。AI **不参与**任何校验判定决策，所有校验结果100%由规则引擎产出，保证可复现、可审计、可追溯。

系统定义了 9 种规则类型，覆盖字段级、记录级、跨数据集级三个校验层次：

```typescript
enum RuleType {
  NOT_NULL = 'not_null',           // 非空校验
  REGEX = 'regex',                 // 格式校验
  LENGTH_RANGE = 'length_range',   // 长度范围
  ENUM_CHECK = 'enum_check',       // 枚举值校验
  UNIQUE = 'unique',               // 唯一性校验
  DATE_FORMAT = 'date_format',     // 日期格式
  VALUE_RANGE = 'value_range',     // 数值范围
  CROSS_FIELD = 'cross_field',     // 跨字段逻辑
  CROSS_TABLE = 'cross_table',     // 跨表关联
}
```

### 6.1.1 规则层级映射

| 规则类型 | 校验层级 | 数据范围 | 复杂度 |
|---------|---------|---------|--------|
| not_null | 字段级 | 单单元格 | 低 |
| regex | 字段级 | 单单元格 | 低 |
| length_range | 字段级 | 单单元格 | 低 |
| enum_check | 字段级 | 单单元格 | 低 |
| date_format | 字段级 | 单单元格 | 低 |
| value_range | 字段级 | 单单元格 | 低 |
| unique | 记录级 | 全 Sheet | 中 |
| cross_field | 记录级 | 单行多字段 | 中 |
| cross_table | 跨数据集级 | 多 Sheet | 高 |

## 6.2 各规则类型参数格式

每种规则类型的 `executable_params`（可执行参数）格式如下：

| 规则类型 | executable_params 结构 | 示例 | 说明 |
|---------|----------------------|------|------|
| `not_null` | `{ placeholders: string[] }` | `{ placeholders: ["无", "-", "待补充", "待补充", "NA", "null"] }` | 工号不允许空值，同时拦截常见占位符 |
| `regex` | `{ pattern: string }` | `{ pattern: "^\\d{8}$" }` | 工号固定 8 位纯数字 |
| `length_range` | `{ min: number, max: number }` | `{ min: 2, max: 15 }` | 姓名长度 2-15 个字符 |
| `enum_check` | `{ values: string[] }` | `{ values: ["晋升", "降职", "平调", "离职"] }` | 异动类型枚举 |
| `unique` | `{ scope: "sheet" \| "global" }` | `{ scope: "sheet" }` | 单表内工号唯一 |
| `date_format` | `{ format: string }` | `{ format: "YYYY-MM-DD" }` | 日期必须符合 YYYY-MM-DD 格式 |
| `value_range` | `{ min: number, max: number, inclusive: boolean }` | `{ min: 0, max: 150, inclusive: true }` | 年龄 0-150，含边界值 |
| `cross_field` | `{ condition: string, thenFields: string[], checkType: RuleType }` | 见下方示例 | 满足条件时字段组执行特定校验 |
| `cross_table` | `{ targetSheet: string, joinField: string, assertPresence: boolean }` | 见下方示例 | 跨表关联存在性校验 |

### 6.2.1 cross_field 规则详解

跨字段规则表达"当某条件满足时，某些字段必须通过某类校验"的业务逻辑。

```typescript
interface CrossFieldParams {
  condition: string;       // 触发条件，格式："字段名 == '值'" 或 "字段名 != '值'"
  thenFields: string[];    // 条件触发后需要校验的字段列表
  checkType: RuleType;     // 对 thenFields 执行的校验类型（通常为 not_null）
}
```

**示例**：异动类型为"晋升"时，"现部门"、"现职级"、"现生效日"三个字段必须非空。

```json
{
  "condition": "异动类型 == '晋升'",
  "thenFields": ["现部门", "现职级", "现生效日"],
  "checkType": "not_null"
}
```

**执行逻辑**：
1. 遍历每一行记录
2. 检查 condition 是否成立（解析条件表达式，对当前行求值）
3. 若条件成立，则对 thenFields 中每个字段执行 checkType 对应的校验
4. 若字段未通过校验，生成一条 ValidationResult

### 6.2.2 cross_table 规则详解

跨表规则表达"A 表的某字段值应当在 B 表中存在"的关联校验逻辑。

```typescript
interface CrossTableParams {
  targetSheet: string;       // 目标 Sheet 名称
  joinField: string;         // 关联字段（在两表中都存在的公共字段）
  assertPresence: boolean;   // true = 必须在目标表中存在；false = 必须不存在
}
```

**示例**：花名册中的工号应在历史晋升信息表中出现。

```json
{
  "targetSheet": "历史晋升信息",
  "joinField": "工号",
  "assertPresence": true
}
```

**执行逻辑**：
1. 收集当前 Sheet 中 joinField 的所有值，构建集合 A
2. 收集 targetSheet 中 joinField 的所有值，构建集合 B
3. 若 assertPresence 为 true，找出 A 中不在 B 中的值
4. 对每个缺失值对应的行，生成 ValidationResult

### 6.3.1 校验结果

```typescript
interface ValidationResult {
  id?: string;               // 数据库主键（插入后生成）

  severity: 'error' | 'warning' | 'info';  // 问题等级
```

### 6.3.2 执行上下文

```typescript
interface ExecutionContext {

  sheetName: string;                         // 当前校验的 Sheet 名称
```

```typescript
interface RuleHandler {
```

## 6.4 规则执行器架构

采用**策略模式（Strategy Pattern）**，每种规则类型由一个独立的 Handler 类实现。所有 Handler 实现统一的 `RuleHandler` 接口，由 Executor 统一分发。

### 6.4.1 目录结构

```
lib/engine/
├── executor.ts         # 入口：接收规则列表 + 数据，分发到对应 Handler
├── batch-runner.ts     # 分批执行控制器
├── checkpoint.ts       # 断点恢复管理
└── rules/
    ├── base.ts         # 规则基类（通用逻辑）
    ├── not-null.ts     # 空值判断 + 占位符识别
    ├── regex.ts        # 正则表达式匹配
    ├── length-range.ts # 字符串长度范围
    ├── enum-check.ts   # 枚举值比对
    ├── unique-check.ts # 唯一性检测（需全量数据）
    ├── date-format.ts  # 日期格式解析
    ├── value-range.ts  # 数值范围比较
    ├── cross-field.ts  # 多字段联动校验
    └── cross-table.ts  # 跨表关联校验
```

### 6.4.2 规则基类（base.ts）

所有 Handler 继承自基类，复用通用逻辑：

```typescript
// lib/engine/rules/base.ts

abstract class BaseRuleHandler implements RuleHandler {
  abstract type: RuleType;

  /**
   * 构建 ValidationResult 的通用方法
   */
  protected createResult(
    context: ExecutionContext,
    record: Record<string, any>,
    rowIndex: number,
    fieldName: string | undefined,
    originalValue: string | undefined,
    message: string,
    passed: boolean
  ): ValidationResult {
    return {
      passed,
      severity: context.severity || 'error',
      message,
      ruleId: context.rules[0]?.id || '',
      sheetName: context.sheetName,
      rowIndex,
      fieldName,
      originalValue,
      taskId: context.taskId,
      phase: this.getPhase(),
      batchNumber: 0, // 由 batch-runner 填充
    };
  }

  /**
   * 根据规则类型自动判定所属校验阶段
   */
  protected getPhase(): 'field' | 'record' | 'cross' {
    const fieldTypes: RuleType[] = [
      RuleType.NOT_NULL, RuleType.REGEX, RuleType.LENGTH_RANGE,
      RuleType.ENUM_CHECK, RuleType.DATE_FORMAT, RuleType.VALUE_RANGE,
    ];
    if (fieldTypes.includes(this.type)) return 'field';
    if (this.type === RuleType.UNIQUE || this.type === RuleType.CROSS_FIELD) return 'record';
    return 'cross';
  }

  /**
   * 安全获取字段值（处理 undefined / null）
   */
  protected getField(record: Record<string, any>, fieldName: string): string | undefined {
    const val = record[fieldName];
    if (val === undefined || val === null) return undefined;
    return String(val).trim();
  }

  abstract validate(
    record: Record<string, any>,
    context: ExecutionContext,
    params: Record<string, any>
  ): ValidationResult | null;
}
```

```typescript
// lib/engine/rules/not-null.ts

class NotNullHandler extends BaseRuleHandler {
  type = RuleType.NOT_NULL;

  validate(record: Record<string, any>, context: ExecutionContext, params: Record<string, any>): ValidationResult | null {
    const fieldName = context.rules[0]?.fieldName;
    if (!fieldName) return null;

    const value = this.getField(record, fieldName);
    const placeholders: string[] = params.placeholders || [];

    // 空值判断
    if (value === undefined || value === '') {
      return this.createResult(context, record, 0, fieldName, value,
        `字段「${fieldName}」为空值`, false);
    }

    // 占位符判断
    if (placeholders.includes(value)) {
      return this.createResult(context, record, 0, fieldName, value,
        `字段「${fieldName}」的值为占位符"${value}"`, false);
    }

    return null; // 校验通过
  }
}
```

```typescript
// lib/engine/rules/regex.ts

class RegexHandler extends BaseRuleHandler {
  type = RuleType.REGEX;

  validate(record: Record<string, any>, context: ExecutionContext, params: Record<string, any>): ValidationResult | null {
    const fieldName = context.rules[0]?.fieldName;
    if (!fieldName) return null;

    const value = this.getField(record, fieldName);
    if (value === undefined || value === '') {
      return this.createResult(context, record, 0, fieldName, value,
        `字段「${fieldName}」为空值，无法进行格式校验`, false);
    }

    const pattern = params.pattern;
    if (!pattern) return null;

    try {
      const regex = new RegExp(pattern);
      if (!regex.test(value)) {
        return this.createResult(context, record, 0, fieldName, value,
          `字段「${fieldName}」的值"${value}"不符合格式要求（规则：${pattern}）`, false);
      }
    } catch (e) {
      return this.createResult(context, record, 0, fieldName, value,
        `规则正则表达式无效：${pattern}`, false);
    }

    return null;
  }
}
```

```typescript
class LengthRangeHandler extends BaseRuleHandler {
  type = RuleType.LENGTH_RANGE;

  validate(record: Record<string, any>, context: ExecutionContext, params: Record<string, any>): ValidationResult | null {
    const fieldName = context.rules[0]?.fieldName;
    if (!fieldName) return null;

    const value = this.getField(record, fieldName);
    if (value === undefined) return null; // 空值由 not_null 规则处理

    const { min, max } = params;
    const len = value.length;

        `字段「${fieldName}」的长度为 ${len}，小于最小要求 ${min}`, false);
    }
    if (len > max) {
      return this.createResult(context, record, 0, fieldName, value,
        `字段「${fieldName}」的长度为 ${len}，超过最大限制 ${max}`, false);
    }

    return null;
  }
}
```

```typescript
// lib/engine/rules/unique-check.ts

class UniqueCheckHandler extends BaseRuleHandler {
  type = RuleType.UNIQUE;

  /**
   * 唯一性校验需要全量数据，不按单行执行
   * 由 batch-runner 在 record 阶段统一调用
   */
  validateBatch(records: Record<string, any>[], context: ExecutionContext, params: Record<string, any>): ValidationResult[] {
    const fieldName = context.rules[0]?.fieldName;
    if (!fieldName) return [];

    const results: ValidationResult[] = [];
    const seen = new Map<string, number[]>(); // 值 → 行号列表

    for (let i = 0; i < records.length; i++) {
      const value = this.getField(records[i], fieldName);
      if (value === undefined || value === '') continue;

      if (!seen.has(value)) {
        seen.set(value, []);
      }
      seen.get(value)!.push(i + 1);
    }

    // 找出重复值
    for (const [value, rowIndices] of seen) {
      if (rowIndices.length > 1) {
        for (const rowIndex of rowIndices) {
          results.push(this.createResult(
            context, records[rowIndex - 1], rowIndex, fieldName, value,
            `字段「${fieldName}」的值"${value}"在表中重复出现 ${rowIndices.length} 次（行号：${rowIndices.join(', ')}）`,
            false
          ));
        }
      }
    }

    return results;
  }

  validate(record: Record<string, any>, context: ExecutionContext, params: Record<string, any>): ValidationResult | null {
    // 单行模式不调用此方法
    throw new Error('唯一性校验请使用 validateBatch');
  }
}
```

```typescript
// lib/engine/rules/date-format.ts

class DateFormatHandler extends BaseRuleHandler {
  type = RuleType.DATE_FORMAT;

  validate(record: Record<string, any>, context: ExecutionContext, params: Record<string, any>): ValidationResult | null {
    const fieldName = context.rules[0]?.fieldName;
    if (!fieldName) return null;

    const value = this.getField(record, fieldName);
    if (value === undefined || value === '') return null;

    const format = params.format || 'YYYY-MM-DD';

    if (!this.isValidDate(value, format)) {
      return this.createResult(context, record, 0, fieldName, value,
        `字段「${fieldName}」的值"${value}"不符合日期格式 ${format}`, false);
    }

    return null;
  }

  private isValidDate(value: string, format: string): boolean {
    // 支持的格式：YYYY-MM-DD, YYYY/MM/DD, YYYYMMDD
    const patterns: Record<string, RegExp> = {
      'YYYY-MM-DD': /^\d{4}-\d{2}-\d{2}$/,
      'YYYY/MM/DD': /^\d{4}\/\d{2}\/\d{2}$/,
      'YYYYMMDD': /^\d{8}$/,
    };
    const pattern = patterns[format];
    if (!pattern) return false;
    if (!pattern.test(value)) return false;

    // 进一步验证日期合法性（如 2024-02-30 是不合法的）
    const date = new Date(value.replace(/\//g, '-'));
    return !isNaN(date.getTime());
  }
}
```

```typescript
class ValueRangeHandler extends BaseRuleHandler {
  type = RuleType.VALUE_RANGE;

  validate(record: Record<string, any>, context: ExecutionContext, params: Record<string, any>): ValidationResult | null {
    const fieldName = context.rules[0]?.fieldName;
    if (!fieldName) return null;

    const value = this.getField(record, fieldName);
    if (value === undefined || value === '') return null;

    const num = Number(value);
    if (isNaN(num)) {
      return this.createResult(context, record, 0, fieldName, value,
        `字段「${fieldName}」的值"${value}"不是有效数字`, false);
    }

    const { min, max, inclusive = true } = params;

    if (inclusive) {
      if (num < min || num > max) {
        return this.createResult(context, record, 0, fieldName, value,
          `字段「${fieldName}」的值 ${num} 超出范围 [${min}, ${max}]`, false);
      }
    } else {
      if (num <= min || num >= max) {
        return this.createResult(context, record, 0, fieldName, value,
          `字段「${fieldName}」的值 ${num} 超出范围 (${min}, ${max})`, false);
      }
    }

    return null;
  }
}
```

#### cross-field.ts

```typescript
// lib/engine/rules/cross-field.ts

class CrossFieldHandler extends BaseRuleHandler {
  type = RuleType.CROSS_FIELD;

  validate(record: Record<string, any>, context: ExecutionContext, params: Record<string, any>): ValidationResult | null {
    const { condition, thenFields, checkType } = params;
    if (!condition || !thenFields || !checkType) return null;

    // 解析并评估条件
    if (!this.evaluateCondition(condition, record)) {
      return null; // 条件不满足，跳过此规则
    }

    // 条件成立，校验目标字段
    const handler = this.getHandlerForType(checkType);
    if (!handler) return null;

    const results: ValidationResult[] = [];
    for (const field of thenFields) {
      const value = this.getField(record, field);
      const result = handler.validate(record, {
        ...context,
        rules: [{ ...context.rules[0], fieldName: field }],
      }, params);
      if (result) {
        result.message = `触发条件"${condition}"，但字段「${field}」未通过${checkType}校验`;
        results.push(result);
      }
    }

    // 返回第一个失败结果（或 null）
    return results.length > 0 ? results[0] : null;
  }

  private evaluateCondition(condition: string, record: Record<string, any>): boolean {
    // 简单条件解析器：支持 "字段名 == '值'" 和 "字段名 != '值'"
    const eqMatch = condition.match(/^(.+?)\s*==\s*'(.+?)'$/);
    const neMatch = condition.match(/^(.+?)\s*!=\s*'(.+?)'$/);

    if (eqMatch) {
      const fieldName = eqMatch[1].trim();
      const expectedValue = eqMatch[2];
      return this.getField(record, fieldName) === expectedValue;
    }
    if (neMatch) {
      const fieldName = neMatch[1].trim();
      const unexpectedValue = neMatch[2];
      return this.getField(record, fieldName) !== unexpectedValue;
    }

    return false; // 无法解析的条件默认不触发
  }

  private getHandlerForType(type: RuleType): RuleHandler | null {
    const handlerMap: Record<string, new () => RuleHandler> = {
      [RuleType.NOT_NULL]: NotNullHandler,
      [RuleType.REGEX]: RegexHandler,
      [RuleType.LENGTH_RANGE]: LengthRangeHandler,
      [RuleType.ENUM_CHECK]: EnumCheckHandler,
    };
    const HandlerClass = handlerMap[type];
    return HandlerClass ? new HandlerClass() : null;
  }

  // 基类要求实现，但跨字段规则不使用单行 validate
  validate(record: Record<string, any>, context: ExecutionContext, params: Record<string, any>): ValidationResult | null {
    return this.validate(record, context, params);
  }
}
```

```typescript
  validateBatch(
    sourceRecords: Record<string, any>[],
    context: ExecutionContext,
    params: Record<string, any>
  ): ValidationResult[] {
    const { targetSheet: targetSheetName, joinField, assertPresence } = params;
    if (!targetSheetName || !joinField) return [];

    const targetRecords = context.otherSheets.get(targetSheetName);
    if (!targetRecords || targetRecords.length === 0) {
      // 目标表为空，所有源记录都不满足
      return sourceRecords.map((record, idx) =>
        this.createResult(context, record, idx + 1, joinField,
          this.getField(record, joinField),
          `目标表「${targetSheetName}」中不存在关联数据`, false)
      );
    }

    // 构建目标表 joinField 值的集合
    const targetValues = new Set<string>();
    for (const record of targetRecords) {
      const val = this.getField(record, joinField);
      if (val !== undefined) targetValues.add(val);
    }

    const results: ValidationResult[] = [];
    for (let i = 0; i < sourceRecords.length; i++) {
      const value = this.getField(sourceRecords[i], joinField);
      if (value === undefined) continue;

      const exists = targetValues.has(value);
      if (assertPresence && !exists) {
        results.push(this.createResult(
          context, sourceRecords[i], i + 1, joinField, value,
          `字段「${joinField}」的值"${value}"在「${targetSheetName}」中不存在`,
          false
        ));
      } else if (!assertPresence && exists) {
        results.push(this.createResult(
          context, sourceRecords[i], i + 1, joinField, value,
          `字段「${joinField}」的值"${value}"不应存在于「${targetSheetName}」中`,
          false
        ));
      }
    }

    return results;
  }

  validate(record: Record<string, any>, context: ExecutionContext, params: Record<string, any>): ValidationResult | null {
    throw new Error('跨表校验请使用 validateBatch');
  }
}
```

```typescript
import { NotNullHandler } from './rules/not-null';
import { RegexHandler } from './rules/regex';

import { CrossTableHandler } from './rules/cross-table';

const handlerRegistry: Map<RuleType, RuleHandler> = new Map([
  [RuleType.NOT_NULL, new NotNullHandler()],
  [RuleType.REGEX, new RegexHandler()],
  [RuleType.LENGTH_RANGE, new LengthRangeHandler()],
  [RuleType.ENUM_CHECK, new EnumCheckHandler()],
  [RuleType.UNIQUE, new UniqueCheckHandler()],
  [RuleType.DATE_FORMAT, new DateFormatHandler()],
  [RuleType.VALUE_RANGE, new ValueRangeHandler()],
  [RuleType.CROSS_FIELD, new CrossFieldHandler()],
  [RuleType.CROSS_TABLE, new CrossTableHandler()],
]);

/**
 * 分发规则到对应的 Handler
 */
export function getHandler(rule: ValidationRule): RuleHandler {
  const handler = handlerRegistry.get(rule.executableType);
  if (!handler) {
    throw new Error(`未知的规则类型：${rule.executableType}`);
  }
  return handler;
}

/**
 * 执行单条记录的字段级校验
 */
export function validateRecord(
  record: Record<string, any>,
  rowIndex: number,
  context: ExecutionContext
): ValidationResult[] {
  const results: ValidationResult[] = [];

  for (const rule of context.rules) {
    const handler = getHandler(rule);
    const ctx = { ...context, rules: [rule], severity: rule.severity };
    const result = handler.validate(record, ctx, rule.executableParams);
    if (result) {
      result.rowIndex = rowIndex;
      results.push(result);
    }
  }

  return results;
}
```

## 6.5 分批执行机制

### 6.5.1 批次配置

```typescript
const BATCH_CONFIG = {
  FIELD_LEVEL: 50,        // 字段级：每批 50 条规则
  RECORD_LEVEL: 1000,     // 记录级：每批 1000 行记录
```

### 6.5.2 执行流程

```
启动校验
    ↓
读取最近的 checkpoint（execution_logs 表）
    ↓
无 checkpoint? ──是──→ 从 field 层、batch=0 开始
    │
   否
    ↓
从断点处继续（phase, lastCompletedBatch）
    ↓
┌─────────────────────────────────┐
│  Phase 1: Field-Level（字段级）   │
│  - 按规则分批次（每批50条规则）     │
│  - 每批遍历所有记录执行校验         │
│  - 结果写入 validation_results    │
│  - 保存 checkpoint               │
└──────────────┬──────────────────┘
               ↓
┌─────────────────────────────────┐
│  Phase 2: Record-Level（记录级）  │
│  - 按记录分批次（每批1000行）      │
│  - 每批内执行 unique / cross_field │
│  - 结果写入 validation_results    │
│  - 保存 checkpoint               │
└──────────────┬──────────────────┘
               ↓
┌─────────────────────────────────┐
│  Phase 3: Cross-Level（跨数据集）  │
│  - 按规则分批次（每批10条规则）     │
│  - 每批执行 cross_table 校验       │
│  - 结果写入 validation_results    │
│  - 保存 checkpoint               │
└──────────────┬──────────────────┘
               ↓
         校验完成
```

### 6.5.3 关键设计

**幂等性**：同一批次重跑时，先删除旧结果，再写入新结果。

```typescript
// 删除旧结果
db.delete(validationResults)
  .where(and(
    eq(validationResults.taskId, taskId),
    eq(validationResults.phase, phase),
    eq(validationResults.batchNumber, batchNumber),
  ));

// 写入新结果（批量插入）
db.insert(validationResults).values(newResults);
```

**Checkpoint 内容**：

```typescript
interface Checkpoint {
  taskId: string;
  phase: 'field' | 'record' | 'cross';
  lastCompletedBatch: number;
  lastCompletedRowIndex: number;  // 记录级：最后一行索引
  timestamp: Date;
  status: 'running' | 'completed' | 'failed';
}
```

**进度计算**：

```typescript
function calculateProgress(checkpoints: Checkpoint[]): number {
  const fieldWeight = 0.4;
  const recordWeight = 0.4;
  const crossWeight = 0.2;

  const fieldProgress = getPhaseProgress(checkpoints, 'field');
  const recordProgress = getPhaseProgress(checkpoints, 'record');
  const crossProgress = getPhaseProgress(checkpoints, 'cross');

  return fieldProgress * fieldWeight
       + recordProgress * recordWeight
       + crossProgress * crossWeight;
}
```

**失败重试**：批次失败后自动重试，最多 3 次。

```typescript
async function executeWithRetry(phase: string, batchNumber: number, executor: () => Promise<void>): Promise<boolean> {
  const maxRetries = 3;
  for (let attempt = 1; attempt <= maxRetries; attempt++) {
    try {
      await executor();
      return true;
    } catch (error) {
      console.error(`批次 ${phase}/${batchNumber} 第 ${attempt} 次尝试失败:`, error);
      if (attempt === maxRetries) {
        // 记录失败状态到 checkpoint
        await saveCheckpoint({
          taskId, phase, lastCompletedBatch: batchNumber - 1,
          status: 'failed', timestamp: new Date(),
        });
        return false;
      }
      // 等待后重试（指数退避）
      await new Promise(resolve => setTimeout(resolve, 1000 * attempt));
    }
  }
  return false;
}
```

### 6.5.4 Batch Runner 实现

```typescript
// lib/engine/batch-runner.ts

import { BATCH_CONFIG } from './config';
import { Checkpoint, saveCheckpoint, loadLatestCheckpoint } from './checkpoint';
import { validateRecord, getHandler } from './executor';

interface BatchRunnerOptions {
  taskId: string;
  sheetName: string;
  records: Record<string, any>[];
  otherSheets: Map<string, Record<string, any>[]>;
  fieldMappings: Map<string, string>;
  rules: ValidationRule[];
  onProgress?: (progress: number) => void;
}

export class BatchRunner {
  private options: BatchRunnerOptions;

  constructor(options: BatchRunnerOptions) {
    this.options = options;
  }

  async run(): Promise<ValidationResult[]> {
    // 恢复 checkpoint
    const checkpoint = await loadLatestCheckpoint(this.options.taskId);

    // Phase 1: Field-Level
    await this.runFieldPhase(checkpoint);

    // Phase 2: Record-Level
    await this.runRecordPhase(checkpoint);

    // Phase 3: Cross-Level
    await this.runCrossPhase(checkpoint);

    // 标记完成
    await saveCheckpoint({
      taskId: this.options.taskId,
      phase: 'cross',
      lastCompletedBatch: -1,
      lastCompletedRowIndex: -1,
      timestamp: new Date(),
      status: 'completed',
    });

    return []; // 结果已直接写入数据库
  }

  private async runFieldPhase(checkpoint: Checkpoint | null): Promise<void> {
    const { rules, records, taskId, sheetName, fieldMappings, otherSheets } = this.options;

    // 筛选字段级规则
    const fieldRules = rules.filter(r => r.level === 'field');
    const startBatch = checkpoint?.phase === 'field' ? (checkpoint.lastCompletedBatch + 1) : 0;

    for (let batchIdx = startBatch; batchIdx < fieldRules.length; batchIdx += BATCH_CONFIG.FIELD_LEVEL) {
      const batchRules = fieldRules.slice(batchIdx, batchIdx + BATCH_CONFIG.FIELD_LEVEL);

      await executeWithRetry('field', batchIdx, async () => {
        const results: ValidationResult[] = [];

        for (let i = 0; i < records.length; i++) {
          // 按 fieldMappings 转换列名
          const mappedRecord = this.mapFields(records[i], fieldMappings);
          const ctx: ExecutionContext = {
            taskId, sheetName, records, otherSheets, fieldMappings, rules: batchRules,
          };
          const recordResults = validateRecord(mappedRecord, i + 1, ctx);
          results.push(...recordResults);
        }

        // 幂等写入：先删后插
        // await db.delete(...).where(taskId + phase + batchIdx)
        // await db.insert(...).values(results)
      });

      await saveCheckpoint({
        taskId, phase: 'field', lastCompletedBatch: batchIdx,
        lastCompletedRowIndex: -1, timestamp: new Date(), status: 'running',
      });
    }
  }

  private async runRecordPhase(checkpoint: Checkpoint | null): Promise<void> {
    const { rules, records, taskId, sheetName, fieldMappings, otherSheets } = this.options;

    const recordRules = rules.filter(r => r.level === 'record');
    const startBatch = checkpoint?.phase === 'record' ? (checkpoint.lastCompletedBatch + 1) : 0;

    for (let batchIdx = startBatch; batchIdx < records.length; batchIdx += BATCH_CONFIG.RECORD_LEVEL) {
      const batchRecords = records.slice(batchIdx, batchIdx + BATCH_CONFIG.RECORD_LEVEL);

      await executeWithRetry('record', batchIdx, async () => {
        const results: ValidationResult[] = [];

        for (const rule of recordRules) {
          const handler = getHandler(rule);
          const ctx: ExecutionContext = {
            taskId, sheetName, records: batchRecords, otherSheets, fieldMappings, rules: [rule],
          };

          if ('validateBatch' in handler) {
            const batchResults = (handler as any).validateBatch(
              batchRecords.map(r => this.mapFields(r, fieldMappings)), ctx, rule.executableParams
            );
            results.push(...batchResults);
          }
        }
        // 幂等写入
      });

      await saveCheckpoint({
        taskId, phase: 'record', lastCompletedBatch: batchIdx,
        lastCompletedRowIndex: batchIdx + batchRecords.length,
        timestamp: new Date(), status: 'running',
      });
    }
  }

  private async runCrossPhase(checkpoint: Checkpoint | null): Promise<void> {
    const { rules, records, taskId, sheetName, fieldMappings, otherSheets } = this.options;

    const crossRules = rules.filter(r => r.level === 'cross');
    const startBatch = checkpoint?.phase === 'cross' ? (checkpoint.lastCompletedBatch + 1) : 0;

    for (let batchIdx = startBatch; batchIdx < crossRules.length; batchIdx += BATCH_CONFIG.CROSS_LEVEL) {
      const batchRules = crossRules.slice(batchIdx, batchIdx + BATCH_CONFIG.CROSS_LEVEL);

      await executeWithRetry('cross', batchIdx, async () => {
        const results: ValidationResult[] = [];

        for (const rule of batchRules) {
          const handler = getHandler(rule);
          const ctx: ExecutionContext = {
            taskId, sheetName, records, otherSheets, fieldMappings, rules: [rule],
          };

          if ('validateBatch' in handler) {
            const batchResults = (handler as any).validateBatch(
              records.map(r => this.mapFields(r, fieldMappings)), ctx, rule.executableParams
            );
            results.push(...batchResults);
          }
        }
        // 幂等写入
      });

      await saveCheckpoint({
        taskId, phase: 'cross', lastCompletedBatch: batchIdx,
        lastCompletedRowIndex: -1, timestamp: new Date(), status: 'running',
      });
    }
  }

  private mapFields(record: Record<string, any>, mappings: Map<string, string>): Record<string, any> {
    const mapped: Record<string, any> = {};
    for (const [dataCol, ruleField] of mappings) {
      if (record[dataCol] !== undefined) {
        mapped[ruleField] = record[dataCol];
      }
    }
    return mapped;
  }
}
```

## 6.6 规则解析固化

AI 解析规则后生成的 `executableParams` 必须经过严格验证才能入库固化，防止脏参数导致运行时错误。

### 6.6.1 验证流程

```
AI 生成 executableParams
    ↓
1. JSON 格式校验
    ↓
2. 类型特定验证
   ├── regex → 正则表达式合法性
   ├── value_range → min <= max
   ├── enum_check → values 数组非空
   └── cross_table → targetSheet 存在于素材中
    ↓
3. 验证通过 → 固化到 validation_rules 表
   验证失败 → 标记 confidence = "low"，提示人工修正
```

### 6.6.2 验证器实现

```typescript
// lib/engine/rule-validator.ts

interface ValidationResult {
  valid: boolean;
  errors: string[];
}

export function validateExecutableParams(
  ruleType: RuleType,
  params: Record<string, any>,
  availableSheets?: string[]
): ValidationResult {
  const errors: string[] = [];

  switch (ruleType) {
    case RuleType.NOT_NULL:
      if (!params.placeholders || !Array.isArray(params.placeholders)) {
        errors.push('not_null 规则需要 placeholders 数组');
      }
      break;

    case RuleType.REGEX:
      if (!params.pattern || typeof params.pattern !== 'string') {
        errors.push('regex 规则需要 pattern 字符串');
      } else {
        try {
          new RegExp(params.pattern);
        } catch {
          errors.push(`正则表达式无效：${params.pattern}`);
        }
      }
      break;

    case RuleType.LENGTH_RANGE:
      if (typeof params.min !== 'number' || typeof params.max !== 'number') {
        errors.push('length_range 需要 min 和 max 数字');
      } else if (params.min > params.max) {
        errors.push(`length_range 的 min(${params.min}) 不能大于 max(${params.max})`);
      } else if (params.min < 0) {
        errors.push('length_range 的 min 不能为负数');
      }
      break;

    case RuleType.ENUM_CHECK:
      if (!params.values || !Array.isArray(params.values) || params.values.length === 0) {
        errors.push('enum_check 需要非空的 values 数组');
      }
      break;

    case RuleType.UNIQUE:
      if (!params.scope || !['sheet', 'global'].includes(params.scope)) {
        errors.push('unique 规则需要 scope 参数（sheet 或 global）');
      }
      break;

    case RuleType.DATE_FORMAT:
      if (!params.format || typeof params.format !== 'string') {
        errors.push('date_format 需要 format 字符串');
      }
      break;

    case RuleType.VALUE_RANGE:
      if (typeof params.min !== 'number' || typeof params.max !== 'number') {
        errors.push('value_range 需要 min 和 max 数字');
      } else if (params.min > params.max) {
        errors.push(`value_range 的 min(${params.min}) 不能大于 max(${params.max})`);
      }
      break;

    case RuleType.CROSS_FIELD:
      if (!params.condition || typeof params.condition !== 'string') {
        errors.push('cross_field 需要 condition 字符串');
      }
      if (!params.thenFields || !Array.isArray(params.thenFields) || params.thenFields.length === 0) {
        errors.push('cross_field 需要非空的 thenFields 数组');
      }
      if (!params.checkType || !Object.values(RuleType).includes(params.checkType)) {
        errors.push(`cross_field 的 checkType 无效：${params.checkType}`);
      }
      break;

    case RuleType.CROSS_TABLE:
      if (!params.targetSheet || typeof params.targetSheet !== 'string') {
        errors.push('cross_table 需要 targetSheet 字符串');
      } else if (availableSheets && !availableSheets.includes(params.targetSheet)) {
        errors.push(`目标表「${params.targetSheet}」不存在于素材中`);
      }
      if (!params.joinField || typeof params.joinField !== 'string') {
        errors.push('cross_table 需要 joinField 字符串');
      }
      break;

    default:
      errors.push(`未知的规则类型：${ruleType}`);
  }

  return { valid: errors.length === 0, errors };
}
```

### 6.6.3 固化 API

```typescript
// /api/rules/solidify

export async function POST(req: Request) {
  const { rules, availableSheets } = await req.json();
  const solidifiedRules: ValidationRule[] = [];
  const rejectedRules: { rule: any; errors: string[] }[] = [];

  for (const rule of rules) {
    const validation = validateExecutableParams(
      rule.executableType,
      rule.executableParams,
      availableSheets
    );

    if (validation.valid) {
      // 入库
      const savedRule = await db.insert(validationRules).values({
        ...rule,
        status: 'active',
        createdAt: new Date(),
        updatedAt: new Date(),
      }).returning();
      solidifiedRules.push(savedRule[0]);
    } else {
      // 标记为低置信度，返回错误详情
      rejectedRules.push({ rule, errors: validation.errors });
    }
  }

  return Response.json({
    solidified: solidifiedRules,
    rejected: rejectedRules,
    summary: {
      total: rules.length,
      passed: solidifiedRules.length,
      failed: rejectedRules.length,
    },
  });
}
```
